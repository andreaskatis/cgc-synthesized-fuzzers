Some observations about the trace based on the spear specification
found in /stop/stop.spear ..

- xindex is always 0
  => even when (cmd == 4)

  This seems to violate the "write_key" requirement.

- xact is only ever 3 or 0

  This seems to violate the "write_cmd" requirement.

- xvalue is 0 a lot
  (perhaps writing xreg?)

  Except I would expect to see an xreg value when (cmd == DO_WRITE_K) ie: (cmd == 2) .. it seems to happen when (cmd == 5)
  However, when (cmd == 5) I would expect to see an xvalue sequence of 0 then 1 (per twinkle_cmd_1 and twinkle_cmd_2)
  -> But I don't see xvalue == 1 in the trace.
